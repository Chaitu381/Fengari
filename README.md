# Fengari

Fengari is a product-focused software studio building scalable web platforms, SaaS products, AI-powered systems, automation tools, and modern digital experiences.

We work on products that solve real operational problems for businesses, institutes, and modern teams.

## What We Build

- Custom web applications
- SaaS platforms
- AI-powered systems
- Automation tools
- Learning and assessment platforms
- Attendance and analytics systems
- Business dashboards
- E-commerce admin platforms
- Digital experience websites

## Products

### ScholarLearn

A learning and assessment platform for institutes with online tests, coding assessments, attendance tracking, student analytics, reports, and progress insights.

### BeThere

An AI-assisted face recognition attendance system built for fast verification, accurate records, and institutional reporting.

### StayOps

A PG and hostel management platform for admissions, room occupancy, rent tracking, maintenance, staff work, and owner dashboards.

### Roam

A safety-first ride-hailing and mobility platform focused on verified drivers, transparent pricing, live tracking, and dependable travel.

### Fiji Web Hub

A website and digital experience studio for business websites, landing pages, interactive interfaces, SEO-ready pages, and conversion-focused digital presence.

### PS-Value Mart

A full-stack e-commerce admin platform for managing products, inventory, pricing, images, and store operations.
